3.1 Understanding Files And Directories



Everything stored on your hard drive is saved as a file, though we often think of them in
terms of files and directories. Files can be program files, data files, directories (which are
special files), and system files that help manage the contents of folders. PHP provides
functions to work with all types of files, but you’ll most often be interacting with text files
containing data.


Note: The terms directory and folder are used interchangeably in this guide and refer to
the same concept.
