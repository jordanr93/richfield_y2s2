Internet Programming as a Prerequisite for Advanced Programming Modules

Internet Programming 621/622 is a prerequisite for Programming 631/632 (Java -
Diploma) and Programming 741/742 (C# - BSc). This ensures that students first gain
experience in web-based programming before advancing to more complex programming
concepts in Java or C#.




Internet Programming 621/622 (PHP & Web Development)

This module introduces students to server-side scripting with PHP, database interactions
using MySQL, session management, security practices, and dynamic web application
development. The skills learned here are crucial for understanding backend logic, data

Why is Internet Programming a Prerequisite?

Web Development Knowledge – Understanding how web applications work is crucial
before handling full-stack or enterprise-level programming.

Database Management – Teaches students how to interact with databases, a
fundamental skill for both Java (JDBC) and C# (ADO.NET/Entity Framework).

Logical Thinking & Problem-Solving – Develops coding skills necessary for tackling more
complex programming structures in Java and C#.

Backend Development Foundation – Before moving into Java desktop applications or C#
enterprise systems, students learn the basics of server-side programming through PHP.

By completing Internet Programming 621/622 first, students are well-prepared to take on
Java (Programming 631/632) or C# (Programming 741/742) and advance their skills in
software development.
