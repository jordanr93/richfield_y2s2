4.17 Connecting To MySQL From PHP


So far, we’ve covered the theory behind relational databases and worked directly with
MySQL using the MySQL command-line tool. Now, it’s time to get your PHP scripts
interacting with MySQL. At the time of writing, PHP offers two main ways to connect to
MySQL databases:

The first option is mysqli (MySQL improved), which is specifically designed for MySQL. It
provides the most complete access to MySQL from PHP, offering both procedural
(function-oriented) and object-oriented interfaces. While it can feel overwhelming due
to the large number of functions and classes available, it’s a great choice if you know
you’ll be working exclusively with MySQL and want to leverage the full power of MySQL
in your PHP scripts.
