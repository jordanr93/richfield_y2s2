4.7 Understanding Relational Databases



In simple terms, a relational database is a system that organizes data into tables and
allows it to be associated or grouped by common attributes. For instance, payroll records
can be grouped by employee, department, or date. In a relational database, data is
typically arranged into tables, with each table divided into rows and columns.

Each row in a table represents a data record, which is a set of related pieces of
information, such as details about a particular person. Each column represents a field,
which is a specific type of data that has the same meaning for every record in the table,
such as "first name" or "age."
