2.3.7 Document Everything

All screen designs should be thoroughly documented for later use by programmers. If a CASE
tool or screen generator is used, designs should be numbered and saved in a hierarchical
structure similar to a menu tree. User-approved sketches and storyboards can also serve as
documentation for the user interface.
