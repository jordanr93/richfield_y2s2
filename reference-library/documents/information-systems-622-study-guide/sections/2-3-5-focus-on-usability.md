2.3.5 Focus on Usability

The user interface should include all necessary tasks, commands, and communications
between users and the system. The opening screen should display the main options, with
each option leading to more detailed choices. The goal is to provide a manageable number of
options that are easy for users to understand. Too many options on one screen can
overwhelm users, while too few can complicate navigation with multiple submenus. A
common strategy is to present the most common choice as a default, but allow users to select
other options if needed.




                                 (Tilley and Rosenblatt, 2024)
