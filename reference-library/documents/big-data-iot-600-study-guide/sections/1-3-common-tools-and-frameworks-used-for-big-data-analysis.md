1.3 Common tools and frameworks used for big data analysis.
Hadoop
An open-source framework called Hadoop was created to handle and store massive datasets in
a distributed manner across computer clusters. It stores data in a distributed file system called
HDFS (Hadoop Distributed File System) and processes data in parallel using the MapReduce
programming methodology.
Features that stand out include the ability to handle unstructured, semi-structured, and
structured data. It is incredibly scalable and fault-tolerant. https://hadoop.apache.org
Apache Spark
Apache Spark is a robust big data analytics framework that is available as open-source software.
Spark processes data in-memory, which makes it much faster than Hadoop. It connects with
Hadoop and Apache Kafka and supports several programming languages, including Python, Java,
and Scala. Important attributes includes: processing data in real time with Spark Streaming, MLlib
provides support for machine learning and Processing capability in both batches and streams.
https://spark.apache.org
Apache Kafka
Kafka is a distributed event streaming platform that is open-source. Because of its ability to
manage real-time data flows, high-throughput event streams may be collected, stored, and
analyzed. It frequently works with other big data tools, such as Spark and Hadoop. Its key features
includes Managing high-velocity data streams and guarantees fault tolerance and message
durability. https://kafka.apache.org
MongoDB
Refers to a NoSQL database created specifically for handling and storing unstructured or semi-
structured data. It is adaptable and scalable for big data situations because it employs a
document-oriented methodology and stores data in a format similar to JSON. The main
advantage of MongoDB is data storage without a schema, high fault tolerance and scalability.
Tableau
Large datasets can be analyzed and shown using Tableau, a potent data visualization tool, as
dashboards, graphs, and charts. It easily integrates with Hadoop and Spark, two big data
platforms. Its advantages include: The drag-and-drop interface makes visualization simple and
the capacity to manage huge datasets from multiple sources.
Amazon Web Services (AWS)
AWS offers a range of cloud-based services, including Amazon EMR (Elastic MapReduce),
Redshift, and Kinesis, for big data processing, storage, and analytics. https://aws.amazon.com
