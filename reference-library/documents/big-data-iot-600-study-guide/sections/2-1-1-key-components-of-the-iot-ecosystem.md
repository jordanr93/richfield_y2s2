2.1.1 Key Components of the IoT Ecosystem

The IoT ecosystem comprises several components that work together to create a seamless flow
of data and operations:

Devices and Sensors
Physical objects that have sensors and actuators built into them gather information from their
surroundings or carry out tasks. Security cameras, fitness trackers, smart thermostats, and
industrial equipment are a few examples.

Connectivity
IoT devices use networks to connect with central systems and with each other. Either cable
(Ethernet) or wireless (Wi-Fi, Bluetooth, Zigbee, or cellular networks like 4G/5G) communication
is possible. Remote device management and real-time data exchange depend on dependable
connectivity.

Data Processing and Analysis
Description: IoT devices process, analyse, and store the data they acquire. Processing can take
place at different levels:
    • Data processed at or close to the devices is known as edge computing.
    • Cloud computing is the use of scalable cloud systems for centralised processing.
Goal: Facilitates automation, predictive analysis, and actionable insights.
