2.4.2 basic building blocks of an IOT system
IoT system gateway acts as a link between cloud computing and Internet of Things devices. It
makes it easier to:
     • Data Pre-Processing and Filtering: By processing data locally, this method lowers the
        amount of data sent to the cloud.
     • Protocol Compatibility: Guarantees device compatibility by supporting a variety of
        communication protocols.
     • Command Transmission: Enables IoT devices to carry out operations using actuators by
        sending control commands to them from the cloud.
By using the right protocols to manage communication between the cloud and field gateways,
implementing a cloud gateway improves the compatibility, scalability, and efficiency of the entire
system.
Big Data and Data Storage in IoT
A data lake is where the enormous volumes of data produced by IoT devices are kept in their
unprocessed state. This data, which is frequently produced in streams or batches, is categorised
as large data because of its high volume, velocity, and variety.
Certain datasets are taken out of the data lake, cleaned, organised, and then put into a large data
warehouse when they are needed for analysis. The construction of structured datasets that aid
in trend and pattern recognition as well as the development of algorithms for control applications
is supported by this warehouse, which is optimised for effective data analysis.
IoT Data Analysis and Machine Learning
Data analysis: Using information from the large data warehouse, analysts produce infographics,
diagrams, and schemes. In order to create more appropriate algorithms for control systems,
these tools aid in the identification of patterns and correlations.
Machine Learning (ML) is frequently used by IoT systems to enhance control applications. To
improve accuracy and efficiency, models are regularly updated and improved using past data
kept in the warehouse.
IoT Control Applications
Actuators receive automated commands from control application based on data analysis. For
instance:
Smart Home Systems: Depending on the weather, windows may open or close automatically.
Agriculture: Watering systems start to irrigate plants when soil sensors sense dryness.
Industrial Monitoring: IoT devices send field engineers alerts in the event of a pre-failure
scenario, while sensors track the condition of the equipment. IoT User Applications
Through software, user apps link Internet of Things users to their devices, enabling control and
monitoring through web or mobile applications. With the help of these apps, users may interact
with smart devices, change settings, and initiate operations as necessary.
