3.5.2 Industrial IoT (IIoT)

In manufacturing and industrial settings, IoT enables automation, real-time monitoring, and
predictive maintenance, improving operational efficiency and reducing downtime.

   IoT Technologies: Sensors monitor machine performance, while edge computing processes
   data locally to reduce latency. LoRaWAN and NB-IoT offer long-range, low-power
   communication ideal for industrial environments.

   Standards: OPC UA (Unified Architecture) is a key standard that ensures devices from
   different vendors can exchange data, promoting interoperability between legacy and modern
   systems.

Example: IoT sensors monitor factory machinery, sending alerts when maintenance is required.
This proactive approach helps prevent unexpected breakdowns. X.509 certificates ensure that
devices are securely authenticated before communicating.
